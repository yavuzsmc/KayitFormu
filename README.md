Bu proje, güvenli ve verimli kullanıcı veri toplama amacıyla tasarlanmış, web tabanlı bir kayıt/giriş formu uygulamasıdır. Kullanıcı arayüzü, temel HTML standartlarına uygun olarak yapılandırılmıştır.
test_xxx.html etiketli dosyalar, kayıt formunun başarılı bir veri girişi sonrasında kullanıcıyı yönlendirdiği işlevsel test sayfalarıdır. Bu yapılar, uygulamanın yönlendirme mekanizmasını gösterir
Uygulamanın ana kayıt ve veri giriş arayüzünü içeren dosya, deneme.html olarak adlandırılmıştır.
